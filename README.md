# Ef-Won Website

A Next.js portfolio website for a backend developer specializing in scalable systems.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Technologies Used

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS

## Features

- Responsive design
- Dark theme with gradient backgrounds
- Portfolio of backend projects
- Services offered
- Contact information
